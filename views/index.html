Request Header Parser Microservice

User Story

I can get the IP address, preferred languages (from header Accept-Language) and system infos (from header User-Agent) for my device.

Example

https://requesting-header-microservice.herokuapp.com/api/whoami

Result

{ipaddress: "::ffff:10.138.92.231", language: "en-US", software: "OS: Windows 10.0, Browswer: Chrome"}
